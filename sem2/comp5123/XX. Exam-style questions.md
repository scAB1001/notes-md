### Q1
Who proposed the Law of Accelerating Returns?
- Bill Gates
- Leonard Kleinrock
- Raymond Kurzweil
- Steve Jobs
#### A
Raymond Kurzweil
### Q2
Which company is credited with launching the first widely used cloud computing service?
- IBM
- Amazon
- Google
- Microsoft
#### A
Amazon
### Q3
Which of the following is a key characteristic of cloud computing?
- Manual scaling
- Fixed resource allocation
- On-demand self-service
#### A
On-demand self-service
### Q4
Which cloud service model provides virtualized computing resources over the internet?
1. Infrastructure as a Service (IaaS)
2. Platform as a Service (PaaS)
3. Software as a Service (SaaS)
4. Function as a Service (FaaS)
#### A
Infrastructure as a Service (IaaS)
### Q5
What is the primary purpose of virtualisation in cloud computing?
1. To increase physical hardware availability
2. To create multiple virtual machines on a single physical machine
3. To reduce software costs
4. To enhance network security
#### A
To create multiple virtual machines on a single physical machine
